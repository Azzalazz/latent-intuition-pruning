# Latent Intuition Pruning

A real-time adaptive search algorithm that prioritizes exploration based on user behavior like hover time or hesitation.

## Features
- Prioritizes higher-interest options based on interaction
- No machine learning or training data needed
- Simulates both adaptive and baseline DFS

## Run
```bash
python main.py           # Runs the adaptive algorithm
python baseline_dfs.py   # Compares with standard DFS
python test_cases.py     # Runs a basic test
```
